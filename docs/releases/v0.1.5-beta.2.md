# JAVM v0.1.5-beta.2

## 概览

- 发布范围：`v0.1.5-beta.1..HEAD`
- 本次版本聚焦于界面提示信息完善，多个入口补充了“需要科学上网”的明确说明，减少使用过程中的预期偏差。
- 同步优化了 Windows 发布流程：预发布版本仅打包 NSIS，稳定版本继续输出 NSIS 与 MSI，降低预发布产物体积与分发复杂度。
- 发布日志模板结构已统一为“新增/优化/修复/兼容性与迁移/已知问题”，便于后续版本保持一致表达。

## 新增

- 在番号搜索、视频链接查找、资源刮削空状态等界面位置新增“需要科学上网”提示，帮助用户在操作前明确网络前置条件。

## 优化

- 调整 GitHub Actions Windows 发布任务，根据 tag 是否为预发布自动选择打包目标。
- 预发布版本只生成 NSIS 安装包，减少不必要构建与上传步骤；稳定版本维持 NSIS + MSI 双产物策略。
- 更新发布日志模板的章节结构与编写约束，提升版本说明一致性。

## 修复

- 修正部分界面提示语不够明确的问题，避免用户在未满足网络条件时误判功能异常。

## 兼容性与迁移

- 无特殊操作要求。

## 已知问题

- 暂无已知问题。
