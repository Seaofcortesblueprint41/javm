# JAVM v0.1.7-beta.1

## 概览

- 发布范围：`v0.1.6..HEAD`
- 回退了上一版本新增的应用内 TS 播放能力，避免本地 `.ts` 和 `.m2ts` 文件在内置播放器中出现播放缓慢与闪退。
- 本地 TS 视频现在统一交给系统默认播放器打开，减少不稳定的播放链路。
- 同步清理不再使用的前端播放器依赖，缩小预发布范围内的冗余实现。

## 新增

- 无面向最终用户的新功能。

## 优化

- 调整媒体库列表与视频详情中的播放入口，软件播放模式下会自动识别本地 TS 文件并直接回退到系统播放器，不再误入应用内播放器窗口。
- 移除前端中不再使用的 `mpegts.js` 依赖，减少播放器页面的额外实现分支。

## 修复

- 修复上一版本中本地 `.ts` 与 `.m2ts` 文件走应用内播放器时可能出现播放速度异常、卡顿甚至闪退的问题。

## 兼容性与迁移

- 无特殊操作要求。

## 已知问题

- 本版本不再提供本地 TS 文件的应用内播放能力；如需播放这类文件，将使用系统默认播放器打开。
