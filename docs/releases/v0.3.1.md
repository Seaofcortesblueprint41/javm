# JAVM v0.3.1

## 概览

- 发布范围：`v0.2.2..v0.3.1`
- 新增 5 个数据源，大幅扩展可刮削网站覆盖。
- 全面增强 Cloudflare 验证处理：多窗口管理、状态细化、误判率降低。
- 新增搜索取消功能、并发数设置和 AI 自动翻译，提升搜索体验与可控性。
- 隐藏用户界面中的具体数据源网站名称，使用通用标识代替，降低合规风险。

## 新增

- **多数据源扩展**：新增 5 个数据源解析器，支持从更多站点抓取视频信息。用户可在设置中自由启用或禁用各个刮削网站。
- **搜索取消功能**：搜索过程中可随时点击停止按钮取消操作，不再需要等待所有数据源返回。
- **搜索并发数设置**：允许用户自定义同时请求的数据源数量（1-10），通过信号量控制并发，避免因高并发触发 IP 封禁或验证。
- **AI 自动翻译**：刮削结果支持通过 AI 提供商自动翻译日语和英语内容，可在设置中启用或禁用，翻译结果在保存前自动处理。
- **最大 WebView 窗口数设置**：用户可配置同时打开的 WebView 窗口数量，避免过多窗口占用资源。
- **搜索结果增强**：搜索结果新增详情页链接、数据丰富度标签和评分展示，优化排序逻辑。

## 优化

- **数据源名称脱敏**：设置页面中的默认刮削网站下拉选项及站点列表，不再显示具体网站名称，统一使用"数据源 1"~"数据源 N"的通用标识。开发者模式下仍可查看站点 ID 以便调试。
- **Cloudflare 验证检测大幅增强**：移除易导致误判的品牌文案硬标记，增加可见文字长度限制，有效降低正常页面被误判为验证页的概率。
- **多网站 CF 验证窗口管理**：同一网站只使用一个 WebView，不同网站创建独立 WebView，默认最多同时存在 3 个窗口，提升刮削稳定性。
- **CF 验证状态细化**：状态从简单的开/关扩展为 idle、active、passed、timeout、failed 五种状态，用户提示信息更清晰。
- **AI 翻译逻辑优化**：引入系统提示构建函数，改进翻译策略，增强对日语和英语内容的识别与处理。
- **公共代码提取**：将多个数据源中重复的选择器函数提取为公共模块，提升可维护性。
- **Windows 安装模式**：NSIS 安装包默认使用当前用户模式安装，无需管理员权限。

## 修复

- 修复部分正常页面被误判为 Cloudflare 验证页的问题。
- 移除资源站点设置中已废弃的 fetchMode 字段，简化配置。

## 兼容性与迁移

- 无特殊操作要求。已保存的用户设置会在启动时自动同步为新的显示名称，无需手动操作。
