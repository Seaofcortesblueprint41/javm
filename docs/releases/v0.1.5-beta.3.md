# JAVM v0.1.5-beta.3

## 概览

- 发布范围：`v0.1.5-beta.2..HEAD`
- 本次版本主要用于验证新的 GitHub Actions 多平台发布流程是否按预期工作。
- 发布工作流已从仅 Windows 构建扩展为 Windows、macOS、Linux 三平台构建。
- 构建前会从上游 `nilaoda/N_m3u8DL-RE` Release 自动下载对应平台二进制，并放入 `src-tauri/bin` 后参与 Tauri 打包。

## 新增

- 新增 GitHub Actions 三平台构建矩阵，覆盖 Windows、macOS、Linux。
- 新增发布阶段的 `N_m3u8DL-RE` 自动下载逻辑，按 runner 平台选择对应产物。

## 优化

- 统一 GitHub Release 上传逻辑，支持收集三平台安装包产物。
- 同步更新发版说明文档，明确三平台打包行为与二进制来源。

## 修复

- 修正原先发布流程只能生成 Windows 安装包的问题。

## 兼容性与迁移

- 无需额外迁移操作。

## 已知问题

- 本版本为测试预发布版本，主要用于验证 CI 构建与 Release 附件是否完整生成。
