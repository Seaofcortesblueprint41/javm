# JAVM v0.1.5-beta.1

## 版本概览

- 本次版本基于 `v0.1.4..HEAD` 之间的全部提交整理。
- 新增发布上下文采集与日志生成链路，便于持续输出结构化发布说明。
- 优化 Windows 发布流程对版本日志的读取方式，减少发布环节手工维护成本。
- 设置界面移除语言选项，简化配置项与使用路径。

## 主要更新

- 功能新增：
  - 新增发布上下文采集能力，可从上一个 tag 到当前 HEAD 自动收集提交信息并生成上下文文件。
  - 新增发布日志模板与配套文档，统一版本日志命名规则和编写流程。
- 功能优化：
  - 调整 GitHub Actions 发布流程，优先读取对应版本日志文件作为发布说明来源。
  - 更新版本升级脚本及相关流程文档，降低发版过程中的人工出错概率。
- 问题修复：
  - 通过流程规范化修复历史上发布信息来源不一致的问题，提升版本说明与实际改动的一致性。

## 使用说明

- 本版本为内测版本（beta），建议先在测试环境验证发布链路与核心功能。
- 常规用户无额外迁移步骤。

## 风险与说明

- 当前提交范围内的改动以发布流程、文档与设置项调整为主，业务功能回归范围相对集中。
- 若已有依赖语言切换配置的外部流程，请留意设置项移除后的影响。
