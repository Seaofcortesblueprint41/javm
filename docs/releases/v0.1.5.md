# JAVM v0.1.5

## 概览

- 发布范围：`v0.1.5-beta.5..HEAD`
- 正式版补齐了应用内对本地 TS 视频的直接播放能力，减少依赖外部播放器的场景。
- 修复了视频信息解析中的字段索引错误，截图、导演等元数据展示与写回更稳定。

## 新增

- 播放器新增对本地 `.ts` 与 `.m2ts` 文件的应用内播放支持，统一复用现有 Plyr 控件体验。
- TS 文件播放失败时会在播放器内给出明确提示，并允许一键回退到系统默认播放器。

## 优化

- 调整播放器初始化流程，按视频类型分别接入原生播放、HLS 与 MPEG-TS，降低不同格式之间的兼容性冲突。
- 正式版沿用现有播放器窗口交互，减少新增格式带来的使用成本。

## 修复

- 修复视频数据解析时的字段索引错误，避免截图、导演等信息读取错位。
- 移除已废弃的剧情字段后，同步清理更新链路，避免编辑或保存视频信息时继续写入无效字段。

## 兼容性与迁移

- 无特殊操作要求。

## 已知问题

- 受 mpegts.js 上游能力限制，静态 TS 文件虽然可以播放，但拖动进度条和精确 seek 可能不稳定。
