# JAVM v0.1.5-beta.4

## 概览

- 发布范围：`v0.1.5-beta.3..HEAD`
- 本次版本继续用于验证 GitHub Actions 多平台多架构发布流程。
- 重点修复了上游 `N_m3u8DL-RE` 资产文件名匹配失败的问题，并更新了 macOS runner 与本地二进制跟踪策略。

## 新增

- 发布 workflow 已覆盖 Windows `x64 / ARM64 / x86`、macOS `x64 / arm64`、Linux `x64 / ARM64`。

## 优化

- 下载 `N_m3u8DL-RE` 时改为按平台架构关键字和扩展名匹配资产，兼容上游带版本号与日期后缀的真实文件名。
- 构建前会清理 `src-tauri/bin` 中旧的 `N_m3u8DL-RE` 文件，避免跨平台资源误打包。
- macOS Intel runner 更新为 GitHub 当前支持的 `macos-15-intel`。

## 修复

- 修复 Windows 发布任务中查找 `win-x64.zip` 失败的问题。
- 修复仓库中已跟踪的 `src-tauri/bin/N_m3u8DL-RE.exe` 可能污染其他平台打包资源的问题。

## 兼容性与迁移

- 无需额外迁移操作。

## 已知问题

- 本版本仍为测试预发布版本，主要用于验证 CI 构建矩阵与 Release 附件生成情况。
